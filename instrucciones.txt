Resultados:

- Obtener el precio de una orden de hamburguesa.
	
Instrucciones:

1. Primero en una terminal en la misma carpeta principal "contenedor",
ejecutar el comando:

- npm install

Para instalar las dependencias.

2. Y luego ejecutar el comando:

- npm start

Para inicializar la aplicacion.

Por defecto se ejecutara en el puerto 3000 del localhost.
